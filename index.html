Projects

CraftConnect

CraftConnect is a web-based platform that connects local manufacturers—such as tailors, bakers, and artisans—with customers seeking customized products for children. The platform allows manufacturers to showcase their work, while customers can browse, request personalized items, and communicate directly through a real-time chat system. Built using Python (Flask), HTML/CSS, JavaScript, SQLite and Socket.io, CraftConnect focuses on enhancing customer-manufacturer interaction and supporting local businesses.
